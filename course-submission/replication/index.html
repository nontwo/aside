Replication Package

Reproduce the Aside course project

This page documents the exact local commands for rebuilding the static app, regenerating the report, auditing the submission, and inspecting the preserved browser extension artifact.

Requirements

  • Node.js 20 or newer
  • npm
  • A Chromium browser for optional extension inspection

Build and verify the static app

npm install
npm run build:submission
npm run audit:submission
npm run smoke:submission

The app is fully static and loads synthetic data from sample_conversations.jsonl. It does not require a backend, API key, private profile, or real conversation log.

Full project check

npm test
npx tsc --noEmit
npm run build
npm run audit:public
npm run build:submission
npm run audit:submission
npm run smoke:submission

Extension artifact

Running npm run build:submission copies the current Aside extension build into course-submission/extension/dist. See the extension artifact page for manual inspection steps.

Source notes are also kept in README.md for reviewers who inspect the repository directly.
