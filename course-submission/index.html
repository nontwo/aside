Follow the Sentence, Not the Thread

Aside is a local-context branching system for long AI conversations. It treats a selected passage as the unit of analysis, builds a small evidence window around it, and turns that local text into a focused follow-up prompt.

Long AI answers are dense: one response can contain definitions, comparisons, caveats, examples, and action items. A normal follow-up often drifts because the model sees the entire thread. Aside asks a narrower question: what if the selected sentence is the object the system must explain?

The demo below exposes the workflow as text data. Select a passage and watch the app segment the source answer, anchor the selection, score terms, build a context window, and stage a branch that can run beside the main reading flow.

Run the simulator Read the report

What the app computes

  1. Sentence anchor Find the selected sentence inside a long source answer.
  2. Local evidence Keep the selected passage plus nearby context, not the whole thread.
  3. Text signal Tokenize the passage and surface high-frequency non-stopword terms.
  4. Branch prompt Generate a reproducible prompt for Ask, Why, or New-tab flows.
