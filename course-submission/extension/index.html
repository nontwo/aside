Extension Artifact

Preserved Aside browser extension

The course website is a static Text as Data demo. This folder preserves the companion Chromium extension artifact so reviewers can inspect the original interaction design separately.

Artifact location

The build artifact is generated at course-submission/extension/dist by npm run build:submission.

Manual inspection

  1. Run npm install.
  2. Run npm run build:submission.
  3. Open chrome://extensions/.
  4. Enable Developer Mode.
  5. Select course-submission/extension/dist with "Load unpacked".
  6. Open ChatGPT, select text in an assistant answer, and try Aside's branch actions.

Privacy note

The public course demo uses only synthetic sample data. The extension can operate on real ChatGPT pages during manual testing, but no private conversations are included in this repository.

Source notes are also kept in README.md for reviewers who inspect the repository directly.
